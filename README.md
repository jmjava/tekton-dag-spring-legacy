# tekton-dag-spring-legacy

Sample Maven Spring Legacy WAR for [tekton-dag](https://github.com/jmjava/tekton-dag) pipeline stacks.

- **Build tool:** maven  
- **Runtime:** spring-legacy (WAR)  
- **Java:** 17  

Use with stack app config: `tool: maven`, `runtime: spring-legacy`, `build-command: "mvn -B clean package -DskipTests"`.
